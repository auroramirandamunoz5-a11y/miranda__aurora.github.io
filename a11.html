CUADRO COMPARATIVO: PÁGINA WEB ESTÁTICA Y DINAMICA

CARACTERISTICAS PAGINA WEB ESTATICA PAGINA WEB DINAMICA
QUE ES Una página web estática es un tipo de sitio web donde el contenido se muestra tal como fue creado, sin cambios automáticos ni interacción con bases de datos en el servidor. Está formada principalmente por archivos como HTML, CSS y a veces JavaScript. Una página dinámica es un tipo de página web creada para mostrar contenido de una colección del CMS (Sistema de gestión de contenido). Te permiten crear sitios web que son capaces manejar de manera eficiente una gran cantidad de contenido, a la vez que mantienen un diseño y una experiencia de usuario uniformes.
CARACTERISTICAS Contenido fijo Cada usuario ve exactamente la misma información, a menos que el desarrollador modifique manualmente el código. No usa base de datos La información está escrita directamente en los archivos de la página. Carga rápida Como el servidor solo envía archivos ya creados, las páginas suelen cargar muy rápido. Mayor seguridad Al no tener conexión con bases de datos ni procesos del lado del servidor, hay menos riesgos de ataques. Fácil de alojar (hosting) Se puede subir a casi cualquier servidor o incluso servicios gratuitos. Contenido variable La información puede cambiar dependiendo del usuario o de los datos almacenados. Uso de bases de datos Guarda y consulta información en bases de datos como MySQL o PostgreSQL. Interactividad Permite funciones como: iniciar sesión comentar comprar productos enviar formularios Actualización automática El contenido puede cambiar sin modificar manualmente cada página.
